# 👩‍💼 Human Resources Analytics SQL Project

## 📌 Project Overview

This project focuses on analyzing HR data using advanced SQL techniques such as:

* Joins
* Window Functions
* CTEs
* Aggregations
* Ranking Functions

The dataset includes employee details, departments, salary history, and shift information.

---

## 🧱 Database Tables Used

* Employee
* Department
* EmployeeDepartmentHistory
* EmployeePayHistory
* Shift

---

## ⚙️ Key Features & Analysis

### 🔹 1. HR Dashboard View

* Created a reusable SQL View for dashboard reporting

### 🔹 2. Active Employees

* Identified employees currently working in departments

### 🔹 3. Highest Paid Employee per Department

* Used RANK() window function

### 🔹 4. Department-wise Average Salary

* Calculated salary insights for each department

### 🔹 5. Salary Ranking

* Ranked employees within departments

### 🔹 6. Top Paid Employees

* Identified top 3 highest earners

### 🔹 7. Salary Growth Analysis

* Used LAG() to compare salary changes

### 🔹 8. Department Transfers

* Found employees with multiple department changes

### 🔹 9. Longest Serving Employee

* Based on Hire Date

### 🔹 10. Salary Expense Analysis

* Department with highest salary cost

### 🔹 11. Shift Analysis

* Employee distribution across shifts

### 🔹 12. Missing Data Detection

* Employees without department

### 🔹 13. Salary vs Department Average

* Compared individual vs average salary

### 🔹 14. Hiring Trends

* Monthly hiring analysis

### 🔹 15. High Salary Night Shift Employees

* Filtered insights

### 🔹 16. Department Transfers Frequency

* Most transferred department

### 🔹 17. Duplicate Salary Detection

* Employees with same salary in department

### 🔹 18. Latest Salary Tracking

* Used ROW_NUMBER()

### 🔹 19. Salary Band Classification

* Low / Medium / High salary groups
